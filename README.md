ส่วนที่ 1 โครงสร้างไฟล์ในโฟลเดอร์ของโปรเจ็กต์
โฟลเดอร์ของโปรเจ็กต์ชื่อ group05 ประกอบด้วย
    โฟลเดอร์ doc/ จัดเก็บรายงานและสไลด์นำเสนอ
    โฟลเดอร์ source/ จัดเก็บไฟล์ที่เกี่ยวข้องในโครงงานทั้งหมด
    โฟลเดอร์sql/ จัดเก็บ salapao_clinic.sql สำหรับสร้างฐานข้อมูล และ insert ข้อมูลเบื้องต้น 

ส่วนที่ 2 ขั้นตอนการติดตั้งเว็บไซต์
1. การติดตั้งโปรแกรมที่จำเป็น
ดาวน์โหลดและติดตั้ง XAMPP
เครือข่ายส่วนตัวเสมือนภายใน (VPN) ของมหาวิทยาลัยศรีนครินทรวิโรฒ
ตรวจสอบการทำงานของเซิร์ฟเวอร์ 10.1.3.40 โดยเปิดเบราว์เซอร์และพิมพ์ http://10.1.3.40/phpmyadmin/index.php
เข้าสู่ phpMyAdmin ด้วย Username: 66102010173 และ Password: 66102010173
2. การตั้งค่าฐานข้อมูล
เข้าสู่ phpMyAdmin
    คลิกที่ปุ่ม "New" เพื่อสร้างฐานข้อมูลใหม่
    ตั้งชื่อฐานข้อมูล เช่น 66102010577 แล้วคลิก "Create"
นำเข้าฐานข้อมูล
    คลิกที่ฐานข้อมูล 66102010577
    เลือกแท็บ "Import" และคลิกปุ่ม "Choose File"
    เลือกไฟล์ SQL (เช่น salapao_clinic.sql) ที่มาพร้อมกับโปรเจกต์
    คลิก "Go" เพื่ออัปโหลดโครงสร้างและข้อมูลเบื้องต้นของฐานข้อมูล
3. การตั้งค่าไฟล์โปรเจกต์
ดาวน์โหลดไฟล์เว็บไซต์
    ดาวน์โหลดหรือคัดลอกไฟล์เว็บไซต์ทั้งหมด (รวมถึงไฟล์ PHP, CSS และโฟลเดอร์อื่น ๆ)
ย้ายไฟล์ไปยังโฟลเดอร์ XAMPP
    ย้ายไฟล์ทั้งหมดไปยังโฟลเดอร์ htdocs ในไดเรกทอรีที่ติดตั้ง XAMPP (เช่น C:\xampp\htdocs\) เช่น C:\xampp\htdocs\final-05\source
4. การทดสอบการทำงานของเว็บไซต์
เปิดเบราว์เซอร์และพิมพ์ เปิดเบราว์เซอร์และพิมพ์ http://10.1.3.40/66102010173/final-05/source/
หากเว็บไซต์แสดงผลถูกต้อง แสดงว่าการติดตั้งสำเร็จ
